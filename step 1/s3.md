# Steps

1 Create an s3 bucket.

2 upload the website files on it. which you downloaded earlier from git hub repo. link is also in the Readme.

3 enable static website hosting under properties tab in the selected s3 bucket.

4 off or disable block public access under permissions tab in s3 bucket.

5 in the same permission tab edit bucket policy.
