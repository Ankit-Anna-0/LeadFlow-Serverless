# Steps

1 before proceeding any further or I can say before creating certificate. We first need to buy a domain name.

2 go to the route 53. in route 53 dashboard go to register domain and enter the domain name which you want to purchase.

3 after that wait few minutes for domain name to get activate 

4 go to the hosted zone section and if you see your domain name their this mean it is ready to use.
