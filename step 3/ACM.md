# Steps

1 go to AWS Certificate Manager them click on request certificate.

2 certificate type choose request a public certificate.

3 enter the domain name which you had purchased earlier and leave all the sitting or options default. click next

4 after creating it certificate status remain pending you need to validate it.

5 in the same page click on create records in route 53.

6 after this your pending validation issue will be resolved.

note: acm certificate should be define in us-east region.
