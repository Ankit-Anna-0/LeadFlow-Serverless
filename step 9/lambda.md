# Steps

1 go to lambda then function then create function.

2 enter the function name, then select node.js 

3 then in permission select use an existing role and select the role which you created earlier. 

4 after lambda function is created next step is to modify code in it, copy the code from the GitHub file replace the emails then deploy it.

5 test your code if it give code 200 mean success. if not you can check the log of lambda in cloud watch.

6 check you receiving email you also geta mail after running test in lambda. something mail is marked as a spam so don't forget to check their also.
