# Steps

1 go to API gateway then create a rest api.

2 give it a name leave rest as default. then click on create.

3 now api is created it time to create api resource now. click create resource, just give it a name then create resource.

4 now time to create method. in method type choose POST then integration type lambda then choose your lambda function. leave rest setting default the create it.

5 now we have to enable CORS. click on enable CORS leave everything default except access-control-allow-origin = *   .

6 save it

7 now click on deploy API. New Stage and give a name dev

8 now you are done.

9 go to stage copy invoke url and test it on postman 

10 id it give you code 200 this mean success
